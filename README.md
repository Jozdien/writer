# Writing with Claude

A working directory for drafting writing with Claude Code.

It contains:

- `.claude/CLAUDE.md` — instructions that shape how Claude drafts and revises here: a two-pass process (expand, then distill), a style guide distilled from Williams & Bizup and Paul Graham, and guidance for matching the voice in `examples/` rather than Claude's defaults.
- `examples/` — reference essays used to calibrate voice, sentence openings, and how much narrative scaffolding to use.

To use it, open the directory in Claude Code and give it an outline, rough notes, or a transcript. The instructions in `CLAUDE.md` do the rest.
